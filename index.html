Conhecimentos

HTML

Eu possuo habilidades em HTML, o que significa que sou capaz de criar e editar páginas da web utilizando essa linguagem de marcação. Com conhecimento em HTML, sou capaz de criar estruturas, adicionar textos, imagens, vídeos e links.

CSS

como profissional da área de tecnologia, possuir conhecimentos em CSS é fundamental para o desenvolvimento de sites e aplicações web. Com o CSS, é possível criar páginas com um design.

Design

Meu conhecimento em design inclui habilidades em software de edição de imagem, tipografia e teoria da cor, que me permitem criar designs coesos e eficazes. Com essas habilidades, sou capaz de criar interfaces visuais atraentes e funcionais, bem como desenvolver e implementar códigos que trazem essas interfaces à vida.

Javascript

Com o JavaScript, sou capaz de criar soluções interativas para websites e aplicações web. Além disso, possuo conhecimento em diversas bibliotecas e frameworks, como React, Angular e Node.js, que me permitem desenvolver aplicações complexas de forma eficiente.

Boostrap

Com meu conhecimento em Bootstrap, sou capaz de criar interfaces de usuário elegantes e responsivas, que funcionam bem em diferentes dispositivos, tamanhos de tela e navegadores.

Git e GitHub

Com conhecimento em Git e GitHub, é possível realizar tarefas como criar repositórios, fazer commits de alterações, criar e gerenciar branches, fazer merge de branches e trabalhar em equipe em projetos de software.
